# Testify

## Install

```console
go get github.com/stretchr/testify
```

## Design Test Function

```code
package yours

import (
  "testing"
  "github.com/stretchr/testify/assert"
)

func TestSomething(t *testing.T) {

  assert.True(t, true, "True is true!")

}
```

## Run Test

```console
go test -v -cover ./...
```
